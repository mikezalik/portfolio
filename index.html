UrbanMoon

Summary

This app utilizes the openweather API to provide geolocal weather data.

Design

In the design process of the application, I started with the idea of simple, straightforward and current conditions being most important. The primary user feature is simply to allow location services in browser and accept when prompted. The intent of the app is to provide robust, straightforward weather information at the user location. The design process took a mobile first approach, as I envisioned that users would be on their mobile device while in use.
